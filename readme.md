# CLONET
## 概要
衣服の管理アプリです。
ご自身の衣服をインターネット上で管理でき、お気に入りのコーデイネートを登録しておくこともできます。
また、他のユーザーの投稿も見ることができ、いいねを送ることもできます。
## 目的
外出自粛によってオシャレを楽しむ機会が減ってしまったのではないかと思いました。もっと楽に気軽にオシャレを楽しんでもらいたいと思い、このサービスを作りました。
## ターゲット
* ファッションが好きな人
* 自分のファッションを他の人にも見て欲しいが、自分自身が着ている写真を投稿するのに抵抗のある人
* 衣服選びを簡単にしたい人
* 大量に服を持っており管理に困っている人
## 使用技術
* PHP
* Laravel6.0
* MySQL
* AWS S3
* Heroku
## 機能一覧
* ユーザー登録、ログイン機能
* 投稿機能
  * 画像投稿
  * タグ付け機能
* 投稿の編集機能
* 投稿の削除機能
* プロフィール編集機能
* いいね機能
